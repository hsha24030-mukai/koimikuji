
# 恋みくじアプリ (Love Fortune App)

ボタンをクリックすることで、今日の恋愛運を占うことができるシンプルなWebアプリケーションです。

## 概要

このアプリは、HTML、CSS、JavaScriptを使用して作成されています。
ピンクを基調とした可愛らしいデザインで、ランダムに4種類の運勢（大吉、中吉、小吉、凶）が表示されます。

## 機能

- **おみくじ機能**: 「おみくじを引く」ボタンを押すと、JavaScriptでランダムに選ばれた運勢が表示されます。
- **レスポンシブデザイン**: シンプルなレイアウトで、直感的に操作が可能です。

## ファイル構成

プロジェクトは以下の3つのファイルで構成されています。

- `index.html`: アプリの構造（タイトル、ボタン、結果表示エリア）を定義。
- `style.css`: 背景色、フォント、ボタンのアニメーションなどのデザインを定義。
- `script.js`: 配列を用いたランダムな運勢選出ロジックを定義。

## 使用技術

- **HTML5**: コンテンツの構造化
- **CSS3**: スタイリング（ホバーエフェクト、角丸など）
- **JavaScript (ES6)**: 乱数生成およびDOM操作

## 使い方

1. すべてのファイルを同じフォルダに保存します。
2. `index.html`をブラウザ（Google Chrome, Microsoft Edge, Safariなど）で開きます。
3. 「おみくじを引く」ボタンをクリックして運勢を確認してください。

## 運勢の種類

表示されるメッセージは以下の通りです：

- **大吉💖**: 素敵な出会いがあるかも！
- **中吉😁**: 少し勇気を出すといいことが起きる
- **小吉😐**: 今日はゆっくり過ごそう
- **凶😵**: 無理せず慎重に行動しよう

## カスタマイズについて

運勢の内容を増やしたい場合は、`script.js`内の `fortunes` 配列に新しいメッセージを追加するだけで簡単に拡張できます。
